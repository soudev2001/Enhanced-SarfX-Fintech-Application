# ATM Images Directory

This directory contains uploaded photos of ATM locations organized by bank code:

```
atms/
├── attijariwafa/
├── boa/
├── banque_populaire/
├── cih/
├── bmci/
└── albarid/
```

## Image Guidelines

- **Format**: JPG, PNG, WebP
- **Max Size**: 2MB per image
- **Max Images**: 5 per ATM
- **Naming**: `{bank_code}_{timestamp}_{original_filename}`

## Directory Structure

Images are automatically organized by bank code when uploaded through the admin interface.
